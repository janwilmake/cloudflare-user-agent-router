Cloudflare Worker Format Demo

How It Works

This Cloudflare Worker demonstrates flexible content delivery in multiple formats. It supports:

  • SEO-friendly HTML for crawlers (automatic detection)
  • Path extensions (.html, .md, .json, .yaml, .png) for explicit format selection
  • Accept header parsing for content negotiation
  • Default to Markdown when no specific format is requested (*/*)
  • OG Image generation for social media sharing
